i.   向左對齊的Label<br>

* 建立6個Button，並使其具有動作事件並有相關動作<br>
i.   New，被按下時印出 開啟新檔<br>

iii. Save，被按下時印出 存檔<br>
iv.  Cut，被按下時印出 剪下<br>
v.   Copy，被按下時印出 複製<br>
vi.  Paste，被按下時印出 貼上使用Grid Bag Layout建立一基本的視窗程式。
